# BinksRouter

Simple .NET based software router for Windows implemented in C# using WPF.

![](docs/screenshots/MainWindow.JPG)

## Features

- Static routing
- ProxyARP implementation
- Dynamic routing using RIPv2 protocol
- Logging to Syslog
